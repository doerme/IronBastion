# 《爆裂守卫\-军事堡垒版》Phaser4 完整开发实现文档

## 一、项目整体重构说明

本项目基于抖音爆款《爆裂守卫》核心玩法，**完全替换原有童话城堡设定为硬核军事堡垒体系**，保留原有的回合对战、抛物线弹道、物理破坏、单位部署核心玩法，整体风格从卡通城堡对战改为**现代军事堡垒攻防战**。

核心替换点：

- 原「城堡」→ 全新**军事堡垒**（多层军事工事、防御掩体、装甲墙体、作战平台）

- 原「英雄」→ **作战兵种/作战单位**（爆破兵、狙击兵、炮兵、工程兵）

- 原「墙体坍塌」→ **军事工事爆破、掩体损毁、装甲击穿、阵地崩塌**

- 原休闲卡通风格 → 写实轻量化军事风格（硝烟、弹痕、爆破冲击、阵地战氛围）

- 玩法逻辑不变，美术、概念、命名、破坏机制、数值全部适配军事堡垒设定

## 二、项目基础信息

### 1\. 项目名称

爆裂守卫：堡垒攻防战（Phaser4 抖音小游戏版）

### 2\. 开发技术栈

- 游戏引擎：Phaser 4（WebGL 2D）

- 开发语言：TypeScript

- 构建工具：Vite

- 运行载体：抖音小程序小游戏 / 网页端

### 3\. 游戏核心定位

轻量化**回合制军事堡垒攻防弹射小游戏**，玩家双方依托己方军事堡垒部署作战兵种，通过抛物线弹道炮击、爆破，摧毁敌方防御工事、作战单位及堡垒核心，极简操作、重度策略、爆破解压。

## 三、游戏世界观与场景设定

### 1\. 世界观

红蓝双方为对立作战阵营，各自构建模块化野战军事堡垒，依托多层防御工事构建阵地。双方轮流部署作战兵种、发起炮火打击，通过爆破敌方掩体、摧毁防御结构、击穿堡垒核心，达成阵地占领、全歼敌方防线的作战胜利目标。

### 2\. 军事堡垒结构设定（核心重构）

摒弃原卡通城堡塔楼结构，采用**标准化野战军事堡垒分层工事结构**，共三层作战阵地，每层功能、防御属性、损毁逻辑完全差异化，贴合军事攻防逻辑。

- **底层：重装防御工事层**

    - 构成：钢筋混凝土掩体、装甲防护墙、防爆工事

    - 特性：血量高、防御高、抗普通炮击，仅爆破兵种可快速破防

    - 作用：堡垒基础承重结构，底层工事大面积损毁会触发上层阵地坍塌

- **中层：主力作战平台层**

    - 构成：作战平台、机枪掩体、物资工事、防御隔板

    - 特性：均衡防御，兼顾承伤与作战部署，是核心兵种部署区域

    - 作用：主力输出、对空对地打击、拦截敌方弹道

- **高层：远程狙击/炮兵阵地层**

    - 构成：高地作战台、狙击掩体、远程火炮阵地

    - 特性：防御薄弱、血量低，无承重能力，但输出站位最优

    - 作用：部署远程兵种，打击敌方底层重装工事和核心堡垒

### 3\. 堡垒核心机制

- 每座军事堡垒包含：**堡垒核心血量、分层防御工事、兵种部署阵地、承重结构判定**

- 底层工事损毁超过60%，触发**阵地连锁坍塌**，中层、高层随机掉落损毁

- 堡垒核心血量清零 / 所有防御工事全部损毁 / 驻守兵种全灭，直接判定战败

## 四、核心玩法体系（军事风重构）

### 1\. 回合作战流程（军事攻防流程）

完全沿用原回合逻辑，名称替换为军事作战术语，更贴合题材：

1. **阵地部署阶段**：己方回合，在堡垒各层阵地部署对应作战兵种，调整攻防阵型

2. **炮火打击阶段**：部署结束，所有已部署兵种自动发射抛物线炮弹/弹药，对敌方堡垒工事、单位发起打击

3. **战场结算阶段**：结算工事损毁、单位伤亡、堡垒核心掉血，切换敌方回合

4. 循环交替，直至一方堡垒彻底损毁，对局结束

单局时长：3\-5分钟，适配碎片化休闲玩法

### 2\. 作战兵种体系（替换原英雄体系）

取消卡通英雄设定，改为四大军事兵种，定位清晰、适配堡垒攻防战术：

- **爆破工兵（破防核心）**

    - 适配站位：底层重装工事阵地

    - 技能特性：爆炸范围大，对装甲、混凝土工事造成**破甲暴击伤害**

    - 战术作用：快速撕开敌方底层防御，触发阵地坍塌

- **突击步兵（均衡输出）**

    - 适配站位：中层作战平台

    - 技能特性：攻速快、伤害稳定，兼顾拆工事、清敌方兵种

    - 战术作用：压制敌方中层作战力量，持续消耗防御血量

- **狙击兵（远程精准打击）**

    - 适配站位：高层高地阵地

    - 技能特性：弹道射程远、精准度高，优先打击敌方高价值兵种和堡垒核心

    - 战术作用：远程收割、精准破核，克制脆皮远程单位

- **野战炮兵（范围轰炸）**

    - 适配站位：中层/高层通用

    - 技能特性：大范围溅射轰炸，无差别损毁大面积工事、群体杀伤兵种

    - 战术作用：清场压制，快速摧毁敌方多层防御体系

### 3\. 物理破坏特效重构（军事向）

替换原卡通碎裂效果，全部改为军事爆破表现：

- 工事被击中：混凝土碎块、装甲弹痕、金属碎片飞溅

- 爆炸效果：黑色硝烟、火光冲击、近距离屏幕震动

- 阵地坍塌：分层工事坠落、碎片堆叠、扬尘特效

- 音效替换：炮击声、爆破轰鸣声、金属碎裂声、阵地坍塌巨响

## 七、核心系统技术实现（Phaser4 适配）

### 1\. 军事堡垒生成系统

通过循环批量生成三层工事块，分层设置不同防御、承重属性，底层开启承重判定，高层关闭承重，模拟真实军事阵地结构。工事块统一开启静态物理碰撞，作为阻挡和打击目标。

### 2\. 兵种部署系统

- 仅**阵地部署阶段**可操作部署

- 不同兵种限制对应部署层数，违规无法放置，强化策略性

- 每个部署点位仅可放置单一兵种，部署后锁定点位

- 部署完成后，所有兵种自动锁定敌方堡垒核心为攻击目标

### 3\. 抛物线炮弹弹道系统

基于Phaser4 Arcade物理引擎，模拟真实炮弹抛物线轨迹，加入重力参数模拟弹道下坠，贴合军事炮击效果，替换原简易弹射逻辑。

```Plain Text
// 军事炮弹抛物线发射核心代码
fireArtillery(soldier: Soldier, targetFort: MilitaryFort) {
  const shell = this.add.sprite(soldier.x, soldier.y, 'shell');
  this.physics.add.existing(shell);

  // 瞄准敌方堡垒中心
  const targetX = targetFort.centerX;
  const targetY = targetFort.centerY;
  const angle = Phaser.Math.Angle.Between(soldier.x, soldier.y, targetX, targetY);
  
  // 模拟炮弹高空抛物线弹道
  const speed = soldier.shellSpeed;
  shell.body.velocity.x = Math.cos(angle) * speed;
  shell.body.velocity.y = Math.sin(angle) * speed - 120;
}
```

### 4\. 工事爆破与坍塌系统

- 炮弹碰撞工事块后，计算**伤害\-防御差值**，扣除工事血量

- 工事血量归零则销毁，播放爆破、碎片、硝烟特效

- 实时统计每层工事损毁率，底层损毁率＞60%，触发上层阵地随机坍塌

- 单次爆破触发屏幕震动、音效反馈，强化军事打击手感

- 所有工事全部损毁后，直接扣除堡垒核心血量，判定战局

### 5\. 回合攻防状态机

状态完全军事术语化，逻辑不变：`READY(备战) → DEPLOY(阵地部署) → ATTACK(炮火打击) → SETTLE(战场结算) → GAMEOVER(战局结束)`

## 九、全套落地游戏数值体系（可直接开发使用）

### 1\. 全局基础固定数值

全局数值统一规范，保障对局节奏稳定、攻防平衡，适配3\-5分钟单局时长：

- 单局最大回合数：12回合（12回合未分胜负则按剩余堡垒核心血量判定优胜）

- 单回合时长：部署阶段15秒、炮火打击结算阶段3秒

- 全局重力参数：600（适配Phaser4抛物线炮弹弹道）

- 屏幕震动参数：单次爆破震动时长80ms、震动强度0\.012

- 阵营初始资源：每局初始解锁3个部署点位，最多可部署6个作战兵种

### 2\. 军事堡垒 \& 工事块数值（分层差异化）

#### 2\.1 堡垒全局核心数值

- 红蓝双方堡垒最大核心血量：**1000**

- 阵地坍塌触发阈值：底层工事损毁率 ≥ 60%

- 全工事损毁额外伤害：所有工事清零后，每回合额外扣除堡垒核心血量80点

- 单座堡垒工事总数量：底层12块、中层10块、高层8块，合计30块

### 4\. 战斗伤害结算公式（核心可直接编码）

#### 4\.1 工事伤害计算公式

最终伤害 = \(兵种基础攻击 × 破甲系数 \- 工事防御值\) × 层级伤害修正比例
最低保底伤害：单次攻击不低于5点，避免不破防无效输出

#### 4\.2 堡垒核心伤害结算规则

- 兵种直接命中核心：全额伤害结算 \+ 狙击兵被动增伤

- 工事坍塌溅射核心：单次坍塌固定扣除核心血量40\-120点（随机区间）

- 全工事损毁惩罚：每回合持续扣除80点核心血量

### 5\. 阵地坍塌概率数值

- 底层损毁60%\-70%：上层坍塌概率35%

- 底层损毁70%\-85%：上层坍塌概率65%

- 底层损毁85%以上：上层坍塌概率100%

- 单次坍塌效果：随机销毁中层/高层2\-4块工事，同时造成核心溅射伤害
